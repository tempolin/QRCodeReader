# Quest対応Web QRコードリーダー

## 概要
Questのブラウザでも動作するWeb版QRコードリーダーの雛形です。

## 特徴
- Quest対応を前提としたUI構成
- 自動で外部URLへ遷移しない安全設計
- HTTPSのみ「開く」を有効化

## 起動方法
1. `npx serve .` を実行
2. ブラウザで表示を確認

## Questでの使い方
- 初回起動時にカメラ許可を行う
- 使い終わったら「停止」を押してカメラを止める

## フォルダ構成
- public/: 静的アセット
- src/: アプリのコード
- styles/: スタイル
- docs/: 設計・運用メモ

## トラブルシュート
- 文字化け: UTF-8（BOMなし）で保存されているか確認
- HTTPS必須: カメラ利用はHTTPSまたはlocalhostが必要
- 権限: ブラウザのカメラ許可を確認
