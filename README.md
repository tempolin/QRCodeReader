# Quest対応Web QRコードリーダー

## 概要
Questのブラウザでも動作するWeb QRコードリーダーです。読み取り結果は表示のみで、外部URLへ自動遷移しません。

## 特徴
- Quest対応を想定した大きめのUI
- 自動遷移なし、httpsのみ「開く」を有効化
- 危険スキームやhttpを警告してブロック

## 起動方法
1. `npx serve .`
2. 表示されたURLをブラウザで開く

## Questでの使い方
- 初回にカメラ許可を行ってください
- 使い終わったら「停止」でカメラを止めます

## フォルダ構成
- `index.html`: 画面の骨組み
- `styles/`: スタイル
- `src/`: ロジック
- `docs/`: 設計と運用メモ
- `public/`: 静的アセット

## トラブルシュート
- 文字化け: すべてUTF-8(BOMなし)で保存されているか確認
- HTTPS必須: カメラ利用はHTTPSかlocalhostのみ
- 権限: ブラウザのカメラ許可設定を確認
